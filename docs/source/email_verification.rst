Email verification
------------------

To enable the email verification, configure the email provider in your ``settings.py`` file.

.. code-block:: python

    EMAIL_VERIFICATION = True
    ACCOUNT_EMAIL_CONFIRMATION_EXPIRE_DAYS = 3
    EMAIL_HOST = ...
    EMAIL_PORT = ...
    EMAIL_HOST_USER = ...
    EMAIL_HOST_PASSWORD = ...
    EMAIL_USE_TLS = ...
    DEFAULT_FROM_EMAIL = ...
